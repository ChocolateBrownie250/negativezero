How to capture mic + call audio together (macOS)
Browsers can record only one input device at a time. To capture both your microphone and the audio coming out of your computer (Zoom, browser, music etc.), you need a virtual loopback device. The cheapest path:
  1. Install BlackHole 2ch — brew install --cask blackhole-2ch or download from existential.audio.
  2. Open Audio MIDI Setup → + → Create Aggregate Device. Tick your built-in mic + BlackHole 2ch. Name it "Mic + BlackHole". Drift correction on the BlackHole row.
  3. In the same app, + → Create Multi-Output Device. Tick your speakers + BlackHole 2ch. Name it "Speakers + BlackHole". Use it as your system output in System Settings → Sound when you want to capture call audio.
  4. In the device picker above pick "Mic + BlackHole".
OBS users: in OBS you already have a similar setup; you can also route a Virtual Camera + Virtual Audio device and pick that here.
